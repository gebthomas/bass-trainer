Success Criteria

Beat Microscope

A user can:

* load a recording
* inspect waveform and onset envelope
* see beat grid and detected attacks
* listen while viewing the analysis

Relative Timing Analyzer

A user can:

* compare bass and song timing
* identify leading/lagging sections
* determine whether timing variation is real or an artifact

Groove Comparator

A user can:

* compare two performances
* identify differences in feel and consistency
* track improvement over time
