# Pocket Lab Experiments

## Experiment 001: Wagon Wheel Timing Perception

### Question

Can a player hear and meaningfully describe subtle timing differences in a real bass performance?

### Context

A full-song Wagon Wheel practice recording was analyzed using fixed-grid, shuffle-grid, and exploratory relative-lag methods. The player compared several excerpts and preferred one that the preliminary analysis labeled as lagging, describing it as more "in the pocket," while another excerpt felt slightly rushed.

### Observations

- Perceived pocket may not correspond to zero timing offset.
- Consistency may matter as much as, or more than, average offset.
- A section measured as delayed may still feel better musically.
- Current lag estimates should be treated as relative indicators, not absolute truth.

### Implications

Pocket Lab should distinguish among:

- timing accuracy,
- timing consistency,
- intentional feel,
- perceived groove.

### Next Steps

- Generate controlled timing-shift examples from the same excerpt.
- Compare -60 ms, -30 ms, 0 ms, +30 ms, and +60 ms bass shifts.
- Pair each audio example with a visual timing representation.
- Record player preference and perception notes.
